# Supporting Resources

Shared resources that support KT AI Coding repositories but are not themselves CLI tools, MCP servers, or skills.

## Repositories

| Repository | Status | Notes |
| --- | --- | --- |
| [`kt-aicoding/.github`](https://github.com/kt-aicoding/.github) | created | Organization profile and shared GitHub presentation assets. |
| [`kt-aicoding/images`](https://github.com/kt-aicoding/images) | created | README image, SVG diagram, banner, logo direction, and visual asset workflow references. |

## Boundary

- Keep reusable public assets, diagrams, and organization presentation resources here.
- Keep executable tooling in `kt-aicoding/cli-tools`, `kt-aicoding/mcp-servers`, or standalone tool repositories.
- Do not commit private account details, proprietary source assets, credentials, or paid-service secrets.
